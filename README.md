- Name : Anything you want it to be called
- OAuth Redirect URLs : http://localhost:3000/auth/callback
- Category : Application Integration
- Client Type : Confidental

Register Your app

- Get ClientID
- Click New Secret and get the Client Secret

Open Program

- Twitch Channel : Input your Username to join your channel's Chat
- ClientID : this is the ClientID you get from registering your app on https://dev.twitch.tv
- Client Secret : This is the Secret Code you generated last after making your Application on https://dev.twitch.tv

Congratulation, You have now connected your Twitch Chat to the Program
